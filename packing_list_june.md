## 📱 3. 通訊 / 手機 App

- [x] **遠傳日本漫遊方案**（✅ 已購買 10G，不夠可 App 加購）→ **抵日再開通**
    - 落地前在飛機上 / 一下機：設定 → 連線 → **行動網路 → 資料漫遊：開啟**
    - 開機後手機會自動連 **NTT docomo**；若沒網 → 重開機或手動選電信商
    - 用漫遊好處：不用換卡、台灣門號照常接 OTP / 通話；用完 App 加購，影片 / 備份回飯店 Wi-Fi 做
- [ ] 出發前**關閉背景流量耗用大戶**（YouTube、IG、Google Photos、LINE 自動下載）
- [ ] **長榮 App** 啟用航班推播（颱風通知最快）
- [ ] **Google Maps 離線地圖（可略過）** — 你已買 10G、導航本身很省流量（每小時約數 MB），加上飯店都有 Wi-Fi，**通常不必下載**
    - 只想當「省電 / 郊區沒訊號」的保險才下載：頭像 → 離線地圖 → 選擇您自己的地圖 → 框住範圍 → 下載
- [ ] **Google 翻譯** 下載日文離線包

> 💡 日本插座跟台灣 2 腳相容，但有些飯店是 3 孔接地插座 → **萬國轉接頭可帶可不帶**，不急
