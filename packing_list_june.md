## 🛡️ 1. 保險／文件（颱風季最優先！）

- [ ] **旅平險 + 旅遊不便險**
    - 必加附約：「**旅程取消**」「**班機延誤**」「**行李延誤**」
    - 颱風成颱前 7 天內常不能再加保 → **盡早辦**
    - 推薦：富邦、國泰、明台、安達；1 人 5 天約 NT$500–1,500
- [ ] **確認信用卡刷機票的「旅遊不便險」條款**並截圖存手機
- [ ] **護照效期** 在 2026/12/09 之後仍有效（出發日 +6 個月）
- [ ] **2 張不同網路信用卡**（建議 JCB + Visa／Master，一張當主、一張當備）
- [ ] **預訂紀錄紙本＋雲端雙備份**：
    - BR166 / BR115 e-Ticket
    - 札幌公園飯店、第一滝本館 訂房確認信
    - JR-EAST Train Reservation 北斗號預約代號
    - 鬼花火 zaiko 票券 QR Code
    - 旅平險保單號碼、保險公司海外電話

## 💴 2. 金錢／支付

- [ ] **日幣現金 ¥30,000**（道南バス、神宮御朱印、老店、自販機）
- [ ] 出發前**打給信用卡公司開海外消費通知**
- [ ] **Google Pay 預先加好信用卡**（Visa Touch / JCB Contactless 便利商店、藥妝、ビックカメラ都能用）
- [ ] 抵達新千歲後立刻買 **Kitaca 實體卡 ¥2,000**（含 ¥500 押金，Day 5 機場退卡）

## 📱 3. 通訊 / 手機 App

- [ ] **遠傳日本漫遊方案**（手機不支援 eSIM，改用漫遊最省事）
    - 出發前 1 週用「遠傳行動客服 App」或打 **449** 查方案、依用量自選
    - 申請時指定 06/05 起算，涵蓋 5 天行程＋緩衝
    - 用漫遊好處：不用換卡、台灣門號照常接 OTP / 通話
    - 用完可 App 加購；飯店都有 Wi-Fi，影片 / 備份回飯店做
- [ ] 出發前**關閉背景流量耗用大戶**（YouTube、IG、Google Photos、LINE 自動下載）
- [ ] **長榮 App** 啟用航班推播（颱風通知最快）
- [ ] **Google Maps 離線地圖** 把札幌＋登別＋新千歲先下載（導航不吃流量）
- [ ] **Google 翻譯** 下載日文離線包
- [ ] **乗換案内 或 NAVITIME for Japan Travel**（北海道交通查詢）
- [ ] **Type-C 充電線**＋**豆腐頭充電器**（行充本身的線可能不夠長）

> 💡 日本插座跟台灣 2 腳相容，但有些飯店是 3 孔接地插座 → **萬國轉接頭可帶可不帶**，不急

## 👕 4. 衣物（6 月北海道 14–22°C）

- [ ] **薄外套／防風夾克**（必備！早晚 14°C，登別山區再 -3°C）
- [ ] **長袖上衣 ×3**（純棉 + 速乾混搭）
- [ ] **長褲 ×1–2**（短褲只白天熱時用）
- [ ] **舒適步行鞋**（每天 1 萬步以上，新鞋先磨合）
- [ ] 內衣褲、襪子 ×5
- [ ] **折傘**（6 月降雨機率低，但有就有）
- [ ] **太陽眼鏡**（新綠期戶外時間多，紫外線比想像強）
- [ ] **帽子**（防曬，可選）

## 💊 5. 藥物 / 健康

- [ ] **個人慢性病用藥**（如有，附英文處方箋影本以防被海關問）
- [ ] 感冒藥、止痛藥、腸胃藥
- [ ] **暈車藥**（特急北斗山區轉彎多）
- [ ] OK 繃、肌肉貼布
- [ ] **口罩 ×3**（登別地獄谷硫磺味重）

## 🧳 6. 隨身雜物（容易漏的）

- [ ] **環保購物袋 ×2**（日本超商袋要錢、買土產裝得下）
- [ ] **泡湯用小毛巾**（Day 4 大湯沼川天然足湯要自帶）
- [ ] 旅行夾鏈袋 ×3–5（裝零錢、票根、藥）
- [ ] 隨身斜背小包（拍照、付錢方便，背包放飯店）
- [ ] 防曬乳 SPF30+
- [ ] **保溫水壺**（札幌＋登別冷熱飲都用得到）
- [ ] ❌ **不要帶銀飾**（登別硫磺會讓銀變黑）
- [ ] ❌ **不要帶厚毛衣 / 雪地裝備**（22°C 用不到）

## 📷 7. 二手相機（如果決定買）

- [ ] **SD 卡 ×2**（≥ 64GB，1 張當備用）
- [ ] **備用電池 ×1**（鬼花火等夜間活動耗電）
- [ ] 充電線 / 充電器（部分機型 USB-C 直充就免帶座充）
- [ ] 鏡頭清潔布、鏡頭蓋繩
- [ ] **出發前在台灣實拍 1–2 次練手感**，不要到日本才第一次按快門
