# 日本六天五夜自由行行程規劃（6 月版・4 人團）
**路線：桃園 → 名古屋（NGO）→ 京都 → 大阪 → 桃園（KIX）**

> 🗓️ **出發日期：6 月（貴船川床流水麵當季）**
> 👥 **團員組成：4 人（1 位 iPhone + 3 位 Android）**

---

## 行程總覽

| 天數 | 城市 | 主要景點與活動 |
|------|------|----------------|
| Day 1 | 名古屋 | 抵達、名古屋港水族館、大須商店街、矢場豬排 |
| Day 2 | 名古屋 | 吉卜力公園（全天）、榮町（Sakae）百貨商圈採買 |
| Day 3 | 名古屋 → 京都 | 搭新幹線前往京都、嵐山（竹林、天龍寺）、錦市場、河原町散策 |
| Day 4 | 京都 → 大阪 | 貴船流水麵（上午）、前往大阪、心齋橋・道頓堀 |
| Day 5 | 大阪 | 日本環球影城（USJ）或 大阪城＋黑門市場 |
| Day 6 | 大阪 → 台灣 | 難波採買、關西機場返台 |

---

## Day 1｜抵達名古屋

**交通：** 桃園機場 → 中部國際機場（NGO・セントレア）→ 名古屋市區（名鐵特急 μSky，約 28–40 分鐘）

### 行程安排

| 時間 | 行程 |
|------|------|
| 06:30–10:30 | ✈️ 桃園 → 中部國際機場（航程約 3 小時） |
| 10:30–11:30 | 入境、領行李、購買 IC Card（manaca/ICOCA）或啟用 JR Pass |
| 11:30–12:15 | 🚄 名鐵 μSky 進名古屋市區（約 28–40 分鐘） |
| 12:15–13:30 | 飯店寄放行李、午餐 |
| 13:30–14:00 | 地鐵名港線到「名古屋港」站 |
| 14:00–17:00 | 🐬 **名古屋港水族館**（虎鯨水槽、海豚表演，建議查當日時刻） |
| 17:00–17:45 | 移動至大須（地鐵名港線轉鶴舞線「大須觀音」站） |
| 17:45–19:00 | 🍖 **矢場味噌豬排** 晚餐 |
| 19:00–21:00 | 🛍️ **大須商店街** 散步、文青雜貨、古著 |
| 21:00– | 回飯店休息 |

### 住宿
- **推薦區域：** JR 名古屋站周邊（隔天前往吉卜力公園、第三天搭新幹線皆便利）

---

## Day 2｜吉卜力公園 × 榮町採買

**交通：** 名古屋 → 愛・地球博紀念公園（地鐵東山線「藤之丘」轉 Linimo 磁浮列車，約 60 分鐘）

### 行程安排

| 時間 | 行程 |
|------|------|
| 08:00–08:30 | 飯店早餐、出發 |
| 08:30–09:30 | 🚇 名古屋站 → 地鐵東山線 → 藤之丘 → Linimo 磁浮 → 愛・地球博紀念公園站（約 60 分鐘） |
| 09:30–12:30 | 🌳 **吉卜力公園**（上半場：吉卜力大倉庫、青春之丘） |
| 12:30–13:30 | 園區內或周邊午餐 |
| 13:30–16:00 | 🌳 **吉卜力公園**（下半場：魔女之谷、DonDoKo 森林、幽靈之里） |
| 16:00–17:00 | 🚇 返回名古屋市區（榮站下車） |
| 17:00–19:30 | 🛍️ **榮町（Sakae）百貨商圈**（松坂屋、PARCO、LACHIC、唐吉訶德） |
| 19:30–21:00 | 🍴 晚餐：世界的山將手羽先 / ひつまぶし（蓬萊軒） |
| 21:00–21:30 | 🌃 名古屋電視塔（MIRAI TOWER）+ Oasis 21 夜景 |
| 21:30– | 回飯店休息 |

> ⚠️ **吉卜力公園門票需在官網提前 2–3 個月搶購**；推薦「大さんぽ券」聯票

### 住宿
- 繼續住名古屋站周邊（隔天直接拉行李搭新幹線）

---

## Day 3｜名古屋 → 京都（嵐山 × 錦市場 × 河原町）

**交通：** 名古屋 → 京都（東海道新幹線 のぞみ，約 35 分鐘）→ 京都市區 → 嵐山（JR 嵯峨野線約 15 分）

### 行程安排

| 時間 | 行程 |
|------|------|
| 07:30–08:30 | 飯店早餐、退房 |
| 08:30–09:00 | 🚖 至 JR 名古屋站，**將大件行李宅配至京都／大阪飯店**（ヤマト宅急便，隔日達） |
| 09:00–09:40 | 🚄 東海道新幹線 のぞみ／ひかり 至京都（約 35 分鐘） |
| 09:40–10:00 | 京都站置物櫃寄放隨身行李 |
| 10:00–10:30 | 🚆 JR 嵯峨野線至嵯峨嵐山站（約 15 分鐘） |
| 10:30–11:30 | 🎋 **嵐山竹林小徑** + 🌉 **渡月橋** |
| 11:30–12:30 | ⛩️ **天龍寺庭園**（門票 ¥500，世界遺產） |
| 12:30–13:30 | 🍜 嵐山午餐：嵐山よしむら 蕎麥麵 / ％ ARABICA 咖啡 |
| 13:30–14:30 | 🚆 返回京都市區（嵯峨嵐山 → 京都站 → 烏丸） |
| 14:30–17:30 | 🥢 **錦市場**（17–18 點陸續關門，請盡早） |
| 17:30–19:00 | 🏮 **河原町・先斗町** 散策、鴨川河岸 |
| 19:00–20:30 | 🍴 晚餐：おばんざい（京都家常菜）／拉麵 |
| 20:30– | 飯店 Check-in 休息 |

### 住宿
- **推薦區域：** 烏丸・四條河原町周邊（地鐵與公車皆便利，隔天前往貴船較順路）

---

## Day 4｜貴船流水麵 → 大阪

**交通：** 京都市區 → 貴船口（叡山電鐵，約 30 分鐘）→ 京都 → 大阪（JR 京都線新快速約 30 分鐘，或阪急京都線）

### 行程安排

> 🚨 **前一晚務必先將大件行李用ヤマト宅急便宅配至大阪飯店**（隔日達），當天輕裝出門

| 時間 | 行程 |
|------|------|
| 06:30–07:00 | 早起、飯店退房（行李僅留隨身小包） |
| 07:00–07:30 | 🚇 移動至出町柳站 |
| 07:30–08:30 | 🚃 叡山電鐵「出町柳 → 貴船口」（約 30 分鐘）+ 京都巴士至貴船 |
| 08:30–10:30 | 排隊等候 **ひろ文** 流水麵（11 點開賣，需早到取號） |
| 10:30–12:00 | 💧 **川床流水麵** 體驗（⚠️ 僅 5～9 月開放） |
| 12:00–13:00 | ⛩️ **貴船神社** 參拜（本宮、奥宮、結社） |
| 13:00–14:30 | 🚃 貴船口 → 出町柳 → 京都站 |
| 14:30–15:30 | 🚄 京都 → 大阪（JR 京都線新快速約 30 分鐘） |
| 15:30–16:30 | 大阪飯店 Check-in（取回宅配行李） |
| 16:30–18:00 | 🛍️ **心齋橋筋商店街** 逛街 |
| 18:00–20:00 | 🍴 **道頓堀** 晚餐：章魚燒（くくる）、大阪燒（美津の）、金龍拉麵 |
| 20:00–21:00 | 📸 固力果霓虹看板、戎橋夜景 |
| 21:00– | 回飯店休息 |

> 💡 **非 5–9 月** 可將上午改為「伏見稻荷大社千本鳥居」（出町柳 → 京阪電車約 20 分鐘）

### 住宿
- **推薦區域：** 難波・心齋橋周邊（隔天 USJ／大阪城皆方便，最後一天搭南海電鐵回關西機場最快）

| 07:30–08:00 | 飯店早餐、出發 |
| 08:00–08:45 | 🚆 難波 → 西九条 → ユニバーサルシティ |
| 08:45–09:00 | 入園排隊（建議 9 點開園前抵達） |
| 09:00–13:00 | 🎢 上半場：超級任天堂世界（瑪利歐賽車）、哈利波特禁忌之旅 |

| 21:00–22:00 | 🚆 返回難波、晚餐／消夜 |

## Day 6｜難波採買 → 返台

**交通：** 難波 → 關西國際機場（KIX）（南海電鐵 rapi:t 特急約 40 分鐘 / 空港急行約 50 分鐘）→ 桃園機場

### 行程安排

| 時間 | 行程 |
|------|------|
| 08:00–09:00 | 飯店早餐、整理行李 |
| 09:00–12:00 | 🛍️ **難波／心齋橋採買**：松本清、大國藥妝、OS藥妝、難波 CITY、なんばパークス、髙島屋 |
| 12:00–13:00 | 🍴 午餐 |
| 13:00–14:00 | 飯店退房、取行李 |
| 14:00–14:30 | 🥟 **551 蓬萊豬肉包**（難波站購買，常溫保存約 4 小時） |
| 14:30–15:30 | 🚆 南海電鐵 rapi:t 特急 / 空港急行至 KIX（40–50 分鐘） |
| 15:30–18:30 | ✈️ Check-in、退稅、過海關、登機 |
| 18:30–21:30 | ✈️ KIX → 桃園機場（航程約 3 小時） |

> ⚠️ 建議起飛前 **3 小時** 抵達機場（國際線退稅 + 行李託運排隊較久）
> 💡 南海 rapi:t 早鳥票建議事先在官網或售票機購買

## 實用資訊

### 票券建議

| 票券 | 使用範圍 | 建議 |
|------|----------|------|
| JR Pass（7 日） | 新幹線（名古屋⇄京都）、JR 在來線、關空特急 HARUKA | 出發前在台灣購買較便宜；本行程跨區使用划算 |
| 近鐵周遊券 | 名古屋⇄京都⇄大阪近鐵線 | JR Pass 替代方案 |
| IC Card（ICOCA / Suica / manaca） | 地鐵、私鐵、公車、便利商店 | 入境後立即儲值，全國通用 |
| 吉卜力公園門票 | ジブリパーク | **官網提前 2–3 個月搶購**（lawson-ticket 抽選） |
| USJ Express Pass | 環球影城快速通關 | KKday／KLOOK 或官網購買 |
| 南海電鐵 rapi:t | 難波⇄關西機場 | 早鳥票約 ¥1,300 |

### 各城市住宿推薦區域

- **名古屋（Day 1–2，2 晚）：** JR 名古屋站周邊
- **京都（Day 3，1 晚）：** 烏丸・四條河原町
- **大阪（Day 4–5，2 晚）：** 難波・心齋橋

### 行李建議

- **Day 3 名古屋→京都：** 大件行李可用「ヤマト宅急便」從名古屋飯店宅配至大阪飯店（隔日達），輕裝玩京都
- **Day 4 京都→大阪：** 若已宅配則早上可輕裝去貴船；否則需先回飯店拿行李再搭車

### 注意事項

1. **吉卜力公園門票** 需提前 2–3 個月在官網（ghibli-park.jp）搶購，務必確認行程後立即購票
2. **貴船川床流水麵** 僅限 **5～9 月**，出發前確認是否當季；旺季排隊 1–2 小時
3. **名古屋水族館** 海豚表演有時刻表，建議提前查詢
4. **嵐山竹林** 建議上午較早抵達，人潮較少且光線最佳
5. **錦市場** 多數店舖 17–18 點關門，抵達京都後盡快前往
6. **USJ** 熱門設施需用 APP 預約整理券或購買 Express Pass
7. **551 蓬萊豬肉包** 機場前再買，常溫保存有限

## ⚠️ 出發前請自行查詢最新資訊

> 本行程內容參考資料截止於 **2025 年 5 月**，實際出發日為 **2026 年**，票價、營業時間、班次、門票政策等可能已有變動，請務必於出發前再次確認。

|------|-------------|

| **JR Pass 價格** | 2023 年 10 月已大幅漲價，未來仍可能調整 |
