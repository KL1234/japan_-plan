# 日本六天五夜自由行行程規劃（10 月版・5 人團）
**路線：桃園 → 名古屋（NGO）→ 奈良 → 大阪 → 桃園（KIX）**

> 🗓️ **出發日期：2026/10/01（四）～ 2026/10/06（二）**
> 👥 **團員組成：5 人（1 位 iPhone + 4 位 Android）**
> 🍂 **季節特色：** 10 月上旬秋高氣爽、氣溫約 18–25°C，天氣最舒服
> ⚠️ **貴船流水麵僅限 5–9 月**，10 月已結束；本版本 **略過京都**，直接由名古屋進奈良再進大阪

---

## Day 1｜抵達名古屋

**交通：** 桃園機場 → 中部國際機場（NGO・セントレア）→ 名古屋市區（名鐵特急 μSky，約 28–40 分鐘）

### 行程安排

| 時間 | 行程 |
|------|------|
| 06:30–10:30 | ✈️ 桃園 → 中部國際機場（航程約 3 小時） |
| 10:30–11:30 | 入境、領行李、購買 IC Card（manaca/ICOCA）或啟用 JR Pass |
| 11:30–12:15 | 🚄 名鐵 μSky 進名古屋市區（約 28–40 分鐘） |
| 12:15–13:30 | 飯店寄放行李、午餐 |
| 13:30–14:00 | 地鐵名港線到「名古屋港」站 |
| 14:00–17:00 | 🐬 **名古屋港水族館**（虎鯨水槽、海豚表演，建議查當日時刻） |
| 17:00–17:45 | 移動至大須（地鐵名港線轉鶴舞線「大須觀音」站） |
| 17:45–19:00 | 🍖 **矢場味噌豬排** 晚餐 |
| 19:00–21:00 | 🛍️ **大須商店街** 散步、文青雜貨、古著 |
| 21:00– | 回飯店休息 |

### 住宿
- **推薦區域：** JR 名古屋站周邊（隔天前往吉卜力公園、第三天搭新幹線皆便利）

| 09:30–12:30 | 🌳 **吉卜力公園**（上半場：吉卜力大倉庫、青春之丘） |

## Day 3｜名古屋 → 奈良餵鹿 → 大阪（10/03 星期六）

**交通：** 名古屋 → 京都（東海道新幹線 のぞみ約 35 分）→ 奈良（近鐵京都線特急約 35 分）→ 大阪難波（近鐵＋阪神直通特急約 40 分）

### 行程安排

> 🚨 **出發前務必將大件行李用ヤマト宅急便從名古屋飯店直送大阪飯店**（隔日達，Day 4 到），當天輕裝出門

| 時間 | 行程 |
|------|------|
| 07:00–08:00 | 飯店早餐、退房 |
| 08:00–08:30 | 🚖 至 JR 名古屋站，**大件行李宅配至大阪飯店**（¥1,850／120cm） |
| 08:30–09:10 | 🚄 東海道新幹線 のぞみ／ひかり 至京都（約 35 分） |
| 09:10–09:30 | 京都站換乘近鐵（京都站八條口近鐵改札） |
| 09:30–10:10 | 🚃 近鐵京都線特急「京都 → 近鐵奈良」（約 35–45 分） |
| 10:10–10:30 | 步行至奈良公園（近鐵奈良站徒步約 10 分） |
| 10:30–12:30 | 🦌 **奈良公園 × 東大寺** <br>・鹿仙貝 ¥200／束，**鞠躬鹿很禮貌但會咬衣服** <br>・東大寺大佛殿 ¥800，大佛高 14.98m，世界遺產 |
| 12:30–13:30 | 🍱 午餐：志津香釜飯（奈良名物）／天極堂葛餅 |
| 13:30–15:00 | ⛩️ **春日大社**（本殿 ¥700）<br>・參道兩側 3,000 座石燈籠、若草山山腳更多鹿 |
| 15:00–15:30 | 返回近鐵奈良站 |
| 15:30–16:30 | 🚄 近鐵奈良 → 大阪難波（阪神直通特急／快速急行，約 40 分） |
| 16:30–17:30 | 大阪飯店 Check-in（取回宅配行李） |
| 17:30–19:00 | 🛍️ **心齋橋筋商店街** 逛街 |
| 19:00–20:30 | 🍴 **道頓堀** 晚餐：章魚燒（くくる）、大阪燒（美津の）、金龍拉麵 |
| 20:30–21:30 | 📸 固力果霓虹看板、戎橋夜景 |
| 21:30– | 回飯店休息 |

### 🦌 奈良小提醒

- 鹿仙貝只能在奈良公園內固定攤販購買（¥200 / 束），外食／零食不要隨意給
- 鞠躬互動：把仙貝舉高，鹿會鞠躬；先鞠躬再給更有禮貌
- **衣物與地圖**要收好，鹿會啃紙張與布料
- 10 月氣候宜人（18–25°C），穿長袖＋薄外套剛好

### 💡 交通票券建議

- 名古屋→京都新幹線：使用 **smartEX App** 線上預訂，刷 IC Card 直接進閘門
- 京都→奈良→大阪：可考慮 **近鐵周遊券（5 日券）¥4,500**，涵蓋本段全部近鐵路線，比個別買划算

### 住宿
- **推薦區域：** 難波・心齋橋周邊（之後 3 晚同一飯店，Day 4 大阪城／Day 5 USJ／Day 6 回關西機場皆方便）

## 實用資訊

### 票券建議

| 票券 | 使用範圍 | 建議 |
|------|----------|------|
| JR Pass（7 日） | 新幹線（名古屋⇄京都）、JR 在來線、關空特急 HARUKA | 本行程新幹線只搭一段（Day 3），整段購買 **不划算**，個別買單程即可 |
| **近鐵周遊券（5 日）** | 京都⇄奈良⇄大阪近鐵線，含近鐵特急 | ¥4,500，Day 3 京都→奈良→大阪一票到底，划算 ⭐ |
| IC Card（ICOCA / Suica / manaca） | 地鐵、私鐵、公車、便利商店 | 入境後立即儲值，全國通用 |
| 吉卜力公園門票 | ジブリパーク | **官網提前 2–3 個月搶購**（lawson-ticket 抽選） |
| USJ Express Pass | 環球影城快速通關 | KKday／KLOOK 或官網購買 |
| 南海電鐵 rapi:t | 難波⇄關西機場 | 早鳥票約 ¥1,300 |

### 各城市住宿推薦區域

- **名古屋（Day 1–2，2 晚）：** JR 名古屋站周邊
- **大阪（Day 3–5，3 晚）：** 難波・心齋橋（同一間飯店住 3 晚，免搬行李）

### 行李建議

- **Day 3 名古屋→大阪（中途遊奈良）：** 大件行李用「ヤマト宅急便」從名古屋飯店直送大阪飯店（隔日達，Day 4 到），當天輕裝搭新幹線＋近鐵去奈良餵鹿、傍晚進大阪飯店
- **住宿 3 晚大阪同飯店（Day 3–5）**：Day 4、Day 5 早上完全不用拉行李

### 注意事項

1. **吉卜力公園門票** 需提前 2–3 個月在官網（ghibli-park.jp）搶購，務必確認行程後立即購票
2. **奈良公園的鹿** 會主動靠近搶食，紙本地圖、票券務必收好；不要手拿多餘仙貝奔跑
3. **名古屋水族館** 海豚表演有時刻表，建議提前查詢
4. **Day 3 行程較緊湊**（新幹線＋近鐵＋奈良＋進大阪），早上 8 點前就要出發
5. **黑門市場** 多數店舖下午陸續關門（16–17 點），建議午餐後立即前往
6. **USJ** 熱門設施需用 APP 預約整理券或購買 Express Pass
7. **551 蓬萊豬肉包** 機場前再買，常溫保存有限
